### 명제

- 진실 혹은 거짓으로 진리를 구분할 수 있는 문장
- 컴퓨터 메모리처럼 항상 참과 거짓의 값을 가진다.
- 여러 개의 명제가 조합 가능

### 연산자로 명제 다루기

- 연산자는 명제를 연산하기 위한 도구 6가지

  - Not ¬ / And ⋀ / Or ⋁ / Exclusive Or ⊕

  - Implication → : 조건 명제 , 함축

    - | p    | q    | p → q |
      | ---- | ---- | ----- |
      | T    | T    | T     |
      | T    | F    | F     |
      | F    | T    | T     |
      | F    | F    | T     |

  - Biconditional ↔ : 두 값이 서로 같을 떄 T 반환

### 풀이 방향

- p,q 정의
- 진리표 작성
- q → p : 역
- ¬p →¬q : 이
- ¬q →¬p : 대우
- 본 명제와 대우는 항상 같은 진리값을 가진다.
